rec::robotino::api2 C++ API documentation

The application programming interface (API2) for Robotino(r) from Festo Didactic permits full access to Robotino's sensors and actors. The library provides a C++ and a C interface. Therefor it can be used best with C++ or C programs. For .Net, Java or others we recommend Robotino's REST-API.

The API2 is available in binary form for Windows and Linux.

For instructions how to install the API2 and how to build C++ programs using API2 refer to the Robotino Wiki.
